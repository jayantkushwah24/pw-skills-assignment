Some of the main features of the html5 are as follows:

1.Imporved Multimedia support:- HTML5 allows for the integration of multimedia elements such as audio ad video directly into web pages without the need for plugins like flash.

2.Canvas elements - HTML5 introduces the <canvas> element, which allows for dyanmic, interactive graphics to be created and manipulated within a page.

3.Geolocation API - HTML5 provides an API for obtaining the user's location , enabling web applications to offer location-based services.

4.Local Storage:- HTML5 introduced the localStorage API ,which allows for the storage of data on the user's device, improving performance and reducing the need for round-trip to the server.

5.New Structural Elements:- HTML5 introduced new semantic elements such as <header>, <body> and <footer> , <nav> and <articles that makes it easier to structure and organize content on a web page .

6.Form Improvements:- HTML5 introduced several improvements to forms, including new inputs types like date,time, and color,as well as new attributes such as reducing the need for required and autofocus .

7.Accessibility improvements:- HTML5 provides better support for accessiblity,including the abliity to provides alternative text for images and improved suppoert for screen readers.
